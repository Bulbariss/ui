## Styling

```css lineNumbers=true
.btn-theme {
  @apply bg-indigo-500  text-white font-bold;
}

.btn-theme:hover {
  @apply bg-indigo-600;
}

.btn-outline {
  @apply border-2 border-indigo-500 text-indigo-500 font-bold;
}

.btn-outline:hover {
  @apply bg-indigo-600;
}
```

| Name           | Type                                           | Default                | Description                                                    |
| -------------- | ---------------------------------------------- | ---------------------- | -------------------------------------------------------------- |
| `type`         | `pill`, `classic`, `halfmoon`, `link`,`custom` | `classic`              | Button type.                                                   |
| `size`         | `sm`,`md`,`lg`                                 | `md`                   | Size of the button.                                            |
| `spinnerSize`  | `sm`,`md`,`lg`                                 | `size` prop value      | Size of the spinner.                                           |
| `spinnerColor` | `string`                                       | `text-white`           | The color of spinner                                           |
| `className`    | `string`                                       |                        | className string                                               |
| `isLoading`    | `boolean`                                      | `false`                | If `true`, the button will show a spinner.                     |
| `isDisabled`   | `boolean`                                      | `isLoading` prop value | If `true`, the button will be disabled.                        |
| `loadingText`  | `boolean`                                      | `false`                | If `true`, the button will show text when isLoading is `true`. |
