## Import

```js lineNumbers=true
import { Spinner } from "@bulbariss/ui";
```

## Usage

```jsx lineNumbers=true live=true
render(
  <div className="flex justify-center items-center h-12">
    <Spinner size="lg" color="border-gray-600" />
  </div>
);
```

| Name        | Type                       | Default           | Description            |
| ----------- | -------------------------- | ----------------- | ---------------------- |
| `color`     | `string`                   | `border-gray-300` | Default input styling. |
| `size`      | `sm`, `md`, `lg`, `custom` | `md`              | Spinner sizes.         |
| `className` | `string`                   |                   | className string.      |
